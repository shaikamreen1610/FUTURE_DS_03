# Student Satisfaction Survey Analysis

A comprehensive **Data Analytics & Visualization project** that analyzes student feedback data to evaluate course performance, teaching quality, engagement level, and overall satisfaction using Python.

---

## Project Objective

The objective of this project is to:

* Analyze **student satisfaction survey data**
* Identify patterns in:

  * Feedback submission behavior
  * Course performance
  * Weighted average ratings
  * Sentiment distribution of feedback
* Provide **actionable insights** to improve academic quality and student experience.

##  Key Insights

* Majority feedback falls under **Neutral sentiment**, indicating improvement scope.
* Some courses show **high configuration but low feedback**, suggesting low engagement.
* Weighted average ratings show a **wide performance gap** across courses.
* Courses like *FYBA, SYBMS, TYBSC* achieved **higher satisfaction scores**.
* Very few negative sentiments – good overall academic environment.
